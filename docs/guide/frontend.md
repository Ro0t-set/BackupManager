title: Frontend
---

The frontend is a React app with Vite and Tailwind.

## Main Commands

```bash
cd frontend
npm install

# development
npm run dev

# production build
npm run build

# local build preview
npm run preview
```

## Structure

- `frontend/src/` — App, pages, components and API services
- `frontend/tailwind.config.js` — Tailwind configuration
- `frontend/vite.config.js` — Vite configuration
