title: Frontend
---

The frontend is a React app powered by Vite and Tailwind.

## Main commands

```bash
cd frontend
npm install

# development
npm run dev

# production build
npm run build

# local preview
npm run preview
```

## Structure

- `frontend/src/` — app, pages, components and API services
- `frontend/tailwind.config.js` — Tailwind config
- `frontend/vite.config.js` — Vite config
